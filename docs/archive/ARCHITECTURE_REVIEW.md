**回答**: ✅ **已明确**

**代码位置**: `processors/history_relation_engine.py`

**算法实现**:
1. **TF-IDF 关键词提取** - 提取新闻关键词
2. **规则实体识别** - 识别人名/地名/机构
3. **余弦相似度计算** - 计算新闻相似度
4. **实体权重加成** - 实体词权重更高

**相似度计算**:

**实体权重**:

| 地名 | 1.5 |
